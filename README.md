# Project 003: Check Even or Odd

**Short Description:**  
This project checks whether a given number is even or odd using Python.

**Skills Learned:**  
- Working with user input  
- Data type conversion  
- Conditional statements  
- Modulus operator (%)

**How to Run:**  
1. Open the `even_or_odd.ipynb` file in Jupyter Notebook.  
2. Execute the cells in order.  
3. Enter any integer to see whether it is even or odd.
